# Sold3rAI

Sold3rAI is a website that teaches people how to solder and includes a built-in AI assistant that can answer questions while you learn.

I made this for Macando by Hack Club, to save up for a new soldering iron, the PineCil, by Pine64.

## What it does
* It explains soldering fundamentals
* Helps beginners understand common mistakes
* Includes an AI assistant for soldering-related questions
* Runs entirely in the broswer
* Provides guidance while you learn

## Why I made it

I wanted to create something that felt more interactive than a normal soldering guide. When you're learning a practical skill, it's easy to get stuck on small problems that aren't covered in a tutorial.

For example:

* Why does my solder look dull?
* Why won't the solder stick to the pad?
* Is my iron too hot?
* How much solder should I actually be using?

Instead of spending time searching for answers, Sold3rAI lets you ask the AI directly.
